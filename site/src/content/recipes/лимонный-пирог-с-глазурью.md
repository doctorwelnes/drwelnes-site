ingredients:
  - name: Мягкий творог маложирный
    amount: 250 г
  - name: Яйцо (С0)
    amount: 3 шт.
  - name: Молоко 2,5%
    amount: 80 мл
  - name: Лимонная цедра
    amount: 5-10 г
  - name: Лимонный сок
    amount: 50 мл (2 лимона)
  - name: Эритрит
    amount: 40 г
  - name: Мука
    amount: 80 г
  - name: Протеин
    amount: 2 мерные ложки (50-60 г)
  - name: Соль
    amount: щепотка
  - name: Разрыхлитель
    amount: 1 ч.л.
  - name: Рикотта
    amount: 150 г
  - name: Лимонный сок
    amount: 25 мл (1 лимон)
  - name: Эритрит
    amount: 25 г
steps:
  - Смешайте творог, яйца, молоко, лимонный сок, цедру и эритрит до однородности.
  - Добавьте муку, протеин, соль и разрыхлитель, тщательно перемешайте.
  - Смажьте форму растительным маслом, вылейте тесто.
  - Выпекайте при 180°C 45 мин (двусторонний нагрев + конвекция).
  - Для глазури смешайте рикотту, лимонный сок и эритрит.
  - Равномерно покройте остывший пирог глазурью.
