ingredients:
  - name: Май
    amount: 60 г
  - name: Творог 5%
    amount: 180 г
  - name: Яйцо (С0)
    amount: 1 целое + 1 белок
  - name: Подсластитель (эритрит)
    amount: по вкусу
  - name: Ванилин
    amount: ½ ч.л.
  - name: Мягкий лаваш
    amount: 1 лист
  - name: Желток
    amount: для смазывания
  - name: Кунжут
    amount: для посыпки
steps:
  - Залейте мак теплой водой на 5 минут, перемешайте до набухания.
  - |+
    Смешайте творог, яйцо с белком, набухший мак, подсластитель и ванилин.

  - Выложите начинку на лаваш, равномерно распределите и сверните рулетом.
  - |+
    Смажьте рулет желтком сверху, посыпьте кунжутом.

  - Выпекайте при 180°C 25 мин (двусторонний нагрев + конвекция).
