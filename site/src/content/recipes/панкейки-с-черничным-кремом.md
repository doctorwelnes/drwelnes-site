ingredients:
  - name: Греческий йогурт
    amount: 300 г
  - name: Куриные яйца (С0)
    amount: 1 шт
  - name: Молоко
    amount: 40 мл
  - name: Подсластитель
    amount: по вкусу
  - name: Цельнозерновая мука
    amount: 50 г
  - name: Разрыхлитель
    amount: 1/2 ч. л.
  - name: Замороженная черника
    amount: 50 г
steps:
  - Смешайте греческий йогурт, яйцо, молоко, подсластитель, муку и разрыхлитель
    до однородной массы.
  - Разогрейте сковороду с антипригарным покрытием, выкладывайте тесто порциями
    и жарьте панкейки по 2-3 минуты с каждой стороны до золотистости.
  - Для крема разморозьте чернику, слегка разомните вилкой и смешайте с
    небольшим количеством йогурта (оставленного).
  - Подавайте панкейки теплыми, сверху выложив черничный крем.
