ingredients:
  - name: Морковь
    amount: 200 г
  - name: Яблоки
    amount: 200 г
  - name: Яйцо (С0)
    amount: 2 шт.
  - name: Мука овсяная
    amount: 100 г
  - name: Разрыхлитель
    amount: ½ ч.л.
  - name: Корица молотая
    amount: ½ ч.л.
  - name: Эритрит
    amount: по вкусу (1-2 ст.л.)
  - name: Соль
    amount: по вкусу
  - name: Мягкий творог маложирный
    amount: 240 г
  - name: Эритрит
    amount: по вкусу (1 ст.л.)
steps:
  - Натрите морковь и яблоки на мелкой терке.
  - Смешайте тертые овощи/фрукты, яйца, овсяную муку, разрыхлитель, корицу,
    эритрит и соль руками.
  - Смажьте форму растительным маслом, выложите тесто.
  - Выпекайте при 180°C 40 мин (двусторонний нагрев + конвекция).
  - Для глазури смешайте творог с эритритом, покройте пирог после остывания.
  - Украсьте свежими ягодами сверху.
